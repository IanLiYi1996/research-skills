# Citation Formats Reference Guide

## APA Style (7th Edition)

### Journal Articles

**Basic Format**:
```
Author, A. A., & Author, B. B. (Year). Title of article. Title of Periodical, volume(issue), page-page. https://doi.org/xx.xxx/xxxxx
```

**Examples**:
```
Smith, J., & Jones, M. (2024). Machine learning in climate science. Nature Climate Change, 14(2), 123-135. https://doi.org/10.1038/s41558-024-12345-6

Wong, L., Martinez, R., & Chen, K. (2023). Neural network interpretability: A systematic review. AI Magazine, 44(3), 245-267.
```

**Special Cases**:
- **2 authors**: Author, A., & Author, B.
- **3-20 authors**: List all, use "&" before last
- **21+ authors**: First 19, ..., last author
- **No DOI**: Use URL if available
- **Advance online publication**: Add "Advance online publication" after journal info

### Books

**Basic Format**:
```
Author, A. A. (Year). Title of book (Edition). Publisher. https://doi.org/xxxxx
```

**Examples**:
```
Johnson, T. (2023). Research methods in psychology (5th ed.). Wiley.

Brown, S., Davis, M., & Wilson, R. (2024). The handbook of cognitive neuroscience. MIT Press. https://doi.org/10.7551/mitpress/12345.001.0001
```

### Book Chapters

**Basic Format**:
```
Author, A. A. (Year). Title of chapter. In E. E. Editor (Ed.), Title of book (pp. xx-xx). Publisher.
```

**Example**:
```
Lee, K. (2023). Deep learning architectures. In A. Thompson (Ed.), Advances in artificial intelligence (pp. 45-78). Springer.
```

### Conference Proceedings

**Basic Format**:
```
Author, A. A. (Year). Title of paper. In E. E. Editor (Ed.), Title of proceedings (pp. xx-xx). Publisher.
```

**Example**:
```
Zhang, Y., & Liu, H. (2024). Attention mechanisms in transformers. In Proceedings of the 2024 Conference on Neural Information Processing Systems (pp. 1234-1245). Curran Associates.
```

## MLA Style (9th Edition)

### Journal Articles

**Basic Format**:
```
Author(s). "Title of Article." Title of Journal, vol. number, no. number, year, pp. xx-xx. DOI or URL.
```

**Examples**:
```
Smith, John, and Mary Jones. "Machine Learning in Climate Science." Nature Climate Change, vol. 14, no. 2, 2024, pp. 123-135. https://doi.org/10.1038/s41558-024-12345-6.

Wong, Linda, et al. "Neural Network Interpretability: A Systematic Review." AI Magazine, vol. 44, no. 3, 2023, pp. 245-267.
```

**Special Cases**:
- **1 author**: Last, First.
- **2 authors**: Last, First, and First Last.
- **3+ authors**: Last, First, et al.

### Books

**Basic Format**:
```
Author. Title of Book. Publisher, year.
```

**Example**:
```
Johnson, Thomas. Research Methods in Psychology. 5th ed., Wiley, 2023.
```

### Book Chapters

**Basic Format**:
```
Author. "Title of Chapter." Title of Book, edited by Editor Name, Publisher, year, pp. xx-xx.
```

**Example**:
```
Lee, Karen. "Deep Learning Architectures." Advances in Artificial Intelligence, edited by Anna Thompson, Springer, 2023, pp. 45-78.
```

## Chicago Style (17th Edition)

### Notes-Bibliography System

**Journal Articles**:
```
Author Last, First. "Article Title." Journal Title volume, no. issue (year): page-page. https://doi.org/xxxxx.
```

**Example**:
```
Smith, John, and Mary Jones. "Machine Learning in Climate Science." Nature Climate Change 14, no. 2 (2024): 123-135. https://doi.org/10.1038/s41558-024-12345-6.
```

**Books**:
```
Author Last, First. Book Title. Place of Publication: Publisher, year.
```

**Example**:
```
Johnson, Thomas. Research Methods in Psychology. 5th ed. Hoboken, NJ: Wiley, 2023.
```

### Author-Date System

**In-text**: (Smith and Jones 2024, 125)

**Reference List**:
```
Smith, John, and Mary Jones. 2024. "Machine Learning in Climate Science." Nature Climate Change 14 (2): 123–135. https://doi.org/10.1038/s41558-024-12345-6.
```

## Harvard Style

### Journal Articles

**Format**:
```
Author, A. and Author, B. (year) 'Title of article', Title of Journal, volume(issue), pp. page-page.
```

**Example**:
```
Smith, J. and Jones, M. (2024) 'Machine learning in climate science', Nature Climate Change, 14(2), pp. 123-135.
```

**3+ authors**: Use 'et al.' after first author

### Books

**Format**:
```
Author, A. (year) Title of book. Edition. Place: Publisher.
```

**Example**:
```
Johnson, T. (2023) Research methods in psychology. 5th edn. Hoboken, NJ: Wiley.
```

## IEEE Style

### Journal Articles

**Format**:
```
[#] A. A. Author, "Title of article," Abbrev. Title of Journal, vol. x, no. x, pp. xxx-xxx, Month year.
```

**Examples**:
```
[1] J. Smith and M. Jones, "Machine learning in climate science," Nat. Clim. Chang., vol. 14, no. 2, pp. 123-135, Feb. 2024.

[2] L. Wong, R. Martinez, and K. Chen, "Neural network interpretability: A systematic review," AI Mag., vol. 44, no. 3, pp. 245-267, Sep. 2023.
```

**4+ authors**: Use "et al." after third author

### Books

**Format**:
```
[#] A. A. Author, Title of Book, xth ed. City, State/Country: Publisher, year.
```

**Example**:
```
[3] T. Johnson, Research Methods in Psychology, 5th ed. Hoboken, NJ: Wiley, 2023.
```

### Conference Papers

**Format**:
```
[#] A. A. Author, "Title of paper," in Proc. Conference Name, City, State/Country, year, pp. xxx-xxx.
```

**Example**:
```
[4] Y. Zhang and H. Liu, "Attention mechanisms in transformers," in Proc. Conf. Neural Inf. Process. Syst., Vancouver, BC, Canada, 2024, pp. 1234-1245.
```

## Common Citation Patterns

### Multiple Works by Same Author

**APA**: Chronological order, earliest first
```
Smith, J. (2022). First work...
Smith, J. (2024). Second work...
```

**MLA**: Alphabetical by title, use three hyphens for repeated author
```
Smith, John. "Early Study..."
---. "Later Study..."
```

### No Author

**APA**: Start with title
```
Climate change impacts report. (2024). Nature...
```

**MLA**: Start with title
```
"Climate Change Impacts Report." Nature...
```

### Online Sources

**APA**: Include retrieval date only if content may change
```
Author, A. (2024). Title. Site Name. Retrieved Month Day, Year, from URL
```

**MLA**: Include access date
```
Author. "Title." Site Name, Date, URL. Accessed Day Month Year.
```

### Secondary Sources

**APA**: Cite both original and secondary
```
Original Author's work, as cited in Secondary Author, year
```

**References**: List secondary source only
